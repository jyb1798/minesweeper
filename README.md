# 지뢰찾기

## 설치

```
npx create-react-app minesweeper
npm install
npm start
```

## 사용 기술

React / Redux

## 구현 내용

- level 선택 가능(Beginner, Intermediate, Expert)
- 버튼을 눌렀을 때 지뢰일 경우 지뢰 표시(boom 이미지)
- 최초의 버튼을 클릭하면 타이머 시작
- 버튼을 눌렀을 때 숫자일 경우 근접한 지뢰의 수 만큼의 숫자 표시
- 오른쪽 클릭을 할 경우 깃발표시, 다시 오른쪽 클릭을 할 경우 깃발이 사라짐
